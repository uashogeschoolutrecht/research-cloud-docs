# Docker + Docker Compose

## Omschrijving
Zet een **reverse proxy** op met **SSL-certificaatvernieuwing** en optionele **OAuth2-authenticatie**.
Automatiseert de installatie van **Docker** en **Docker Compose**, optioneel in rootless-modus per gebruiker.
Wanneer de geselecteerde resource een GPU bevat, wordt ook de **CUDA-toolkit** geïnstalleerd.

:::tip[Aanbeveling]
Startpunt voor onderzoekers of ontwikkelaars die een containerized applicatie willen testen of deployen met netwerktoegang en optionele authenticatie.
:::
