# Windows Server 2019

## Omschrijving
Start een Windows-omgeving die je kunt gebruiken via een Remote Desktop-verbinding of door in te loggen in de browser met je organisatie-credentials.

De omgeving kan naar wens worden uitgebreid met vooraf geïnstalleerde software:
- **LibreOffice** – open-source kantoorsoftware  
- **Microsoft Office** – volledige kantoorsoftware-suite  
- **CUDA** – ondersteuning voor GPU-versnelling  
- **JASP** – open-source statistische analyseomgeving

:::note
Je kunt later ook zelf extra software installeren. Bovenstaande software kun je optioneel selecteren zodat deze direct bij het opstarten van de omgeving al geïnstalleerd is.
:::

:::tip[Aanbeveling]
Gebruik deze omgeving wanneer je werkt met gevoelige data, wanneer je wilt samenwerken met meerdere onderzoekers binnen één omgeving, of wanneer je meer rekenkracht nodig hebt dan je eigen laptop kan bieden.
:::
