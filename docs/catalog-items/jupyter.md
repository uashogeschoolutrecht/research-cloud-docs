# Jupyter Notebook

## Omschrijving

Start een omgeving met [**Jupyterhub**](https://jupyter.org/hub). Via JupyterHub kunnen meerdere gebruikers werken op dezelfde onderzoeksomgeving, waarbij iedere gebruiker na het inloggen via de browser met zijn of haar organisatie-credentials een eigen notebookomgeving krijgt.
In deze omgeving kun je code schrijven, uitvoeren en combineren met tekst, visualisaties en resultaten in interactieve Jupyter Notebooks.

Naast de standaard **Python-kernel** zijn er ook varianten beschikbaar met extra ondersteuning:
- **CUDA** – voor GPU-versnelling van machine learning en numerieke berekeningen  
- **Julia** – voor high-performance numerieke en wetenschappelijke berekeningen  
- **MATLAB** – voor gebruikers die MATLAB-code willen combineren met de interactieve Jupyter-omgeving  
- **R-kernel** – voor statistiek en data-analyse in R  

:::tip[Aanbeveling]
Geschikt voor onderzoek waarbij code, data-analyse en documentatie in één interactieve omgeving worden gecombineerd.
:::
